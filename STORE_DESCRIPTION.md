## Detailed Description

**HyperTabs - Hyprland-Style Tab Management for Power Users**

Transform your browsing with lightning-fast fuzzy search and keyboard-driven navigation. Perfect for developers and power users who manage dozens of tabs daily.

---

**KEY FEATURES**

**Instant Fuzzy Search (Telescope)**
- Search all open tabs by title or URL in milliseconds
- Type `h:` to search browser history
- Type `b:` to search bookmarks
- Smart fuzzy matching tolerates typos

**Quick Access Tabs (Harpoon)**
- Pin important tabs to numbered slots for instant access
- Jump with Alt+1 through Alt+4 from anywhere
- Persistent across browser sessions
- Auto-reopens closed tabs from saved URLs

**Workspace Organization**
- Group related tabs into workspaces
- Switch contexts instantly - work, personal, projects
- Save and restore window layouts
- Hyprland-inspired tab grouping

**Vim-Style Navigation (Optional)**
- Navigate with j/k keys (toggle in settings)
- Arrow keys work by default for everyone
- Full keyboard control - no mouse required

---

**QUICK START**

1. Press `Ctrl+Shift+K` to open HyperTabs
2. Type to search your tabs instantly
3. Use arrow keys to navigate
4. Press Enter to switch tabs
5. Press Escape to close

---

**KEYBOARD SHORTCUTS**

Default:
- `Ctrl+Shift+K` - Open HyperTabs
- `Alt+1` / `Alt+2` - Jump to harpooned tabs

Set additional shortcuts at chrome://extensions/shortcuts:
- `Alt+3` / `Alt+4` - More harpoon slots
- `Ctrl+Shift+M` - Mark tab to harpoon
- `Ctrl+Shift+]` / `[` - Switch workspaces

---

**PRIVACY FIRST**

- All data stored locally on your device
- No tracking, analytics, or data collection
- No external servers or accounts required
- Open source - audit the code yourself

---

**PERFECT FOR**

- Developers with 20+ documentation tabs
- Researchers managing multiple projects
- Anyone who hates clicking through tab bars
- Vim/Neovim users who want familiar keybinds
